# FractalBitcoin

햄쮐 햢햩혪 햡햣햫햣햟혡햦햦 Bitcoin-햟햢햣혜쮏 햫햟 쮐햫쮏쒫 혜햦햪햪햣혝햦혢햫혦혠 혞향쮐쮏 16x16. 햌햟햤햢혦햧 혞향쮐 햣쮏쐄햟향혞햣혝혜혪  256-햠햦혝햫혦햧 햨햩혩혢, 햨쮐쮐혦햧 햦혜쮏혧향혞햣혝혜혪 햢햩혪 혜쮏햢햟햫햦혪 Bitcoin-햟햢햣혜햟.

## 햌햟햨 혨혝 햟햠쮐햟햣혝?

1. 햄쮐 혜쮏햢햟혬혝 혜햦햪햪햣혝햦혢햫혦햧 혞향쮐 16x16 혜 햦혜쮏혧향쮏쒫썛쫧쟳왏 햪햟혝햣햪햟혝햦혢햣혜햨햦혠 혟혞햫햨혡햦햧.
2. 햒향쮐 햣쮏쐄햟향혞햣혝혜혪  256-햠햦혝햫혦햧 햨햩혩혢.
3. 햏햟 쮐햫쮏쒫 햨햩혩혢햟 햡햣햫햣햦혞햣혝혜혪 햦쒫썜햫혦햧 햨햩혩혢 햦 Bitcoin-햟햢햣혜.
4. 햄쮐 쮏쒫왐혪햣혝 햠햟햩햟햫혜 햟햢햣혜햟 혢햣햣향 API.
5. 햇혜햩햦 햠햟햩햟햫혜 햫햣햫혞햩햣쒫쮏, 햠쮐 쮐햟쒫혪햣혝 햦햫혟쮐햪햟혡햦혩 쒫썛  햩햦혢햫혦햣 혜쮏쮏쐄괋왏쫧쟴. 햇혜햩햦 햠햟햩햟햫혜 햫혞햩햣쒫쮏, 햦햫혟쮐햪햟혡햦혪 혞햠햩햦햨혞햣혝혜혪  햨햟햫햟햩햣.
